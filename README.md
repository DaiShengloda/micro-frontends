# micro-frontends（微前端）

基于qiankun从零搭建的一套微前端项目。主应用采用Vue，微应用包括Vue-cli3、react17。

## 📚文献资料

https://qiankun.umijs.org/

## 🚗启动项目

```
$ yarn install
$ yarn mainapp:install
$ yarn mainapp:start
```

Visit `http://localhost:7999`.
